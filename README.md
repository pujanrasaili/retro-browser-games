# 🎮 Retro Browser Games

A collection of classic retro browser games built with React and CSS.  
Clean UI, smooth gameplay, no extra dependencies beyond React.

## 🐍 Games
- **Snake** — Classic snake game with score tracking and increasing speed

## 🚀 Getting Started

```bash
npm install
npm start
```

## 🛠 Tech Stack
- React 18
- CSS3 (no UI libraries)
- Vanilla JS game logic

## 📁 Project Structure
```
src/
  games/
    Snake/
  components/
  App.js
  index.js
```
